# Data-driven_project_with_Leslie_matrix
## Overview
This repository contains the data and Python/Jupyter notebooks used to reproduce the analyses and figures presented in the manuscript "A Data-Driven Classroom Project Using Leslie Matrices." The repository includes the raw data, analysis notebooks, and supporting code needed to reproduce all figures and tables in the manuscript.
## Repository contents

- `data.xlsx` – Dataset used throughout the analyses.
- `Trend analysis (simple linear regression).ipynb`
- `Constructing_Leslie_Dominant_eigenvalue_SAD.ipynb`
- `Sensitivity and Elasticity matrices.ipynb`
- `Impact_of_parameters_Scenarios.ipynb`
- `Projections_Time-dependent_Leslie.ipynb`
- `LICENSE`

## Reproducing the content
Each notebook is independent and can be run separately.
Open the desired notebook in Jupyter Notebook or JupyterLab and execute all cells from top to bottom.
All figures generated by the notebooks are automatically saved in the working directory.
## Notebook descriptions
### Trend analysis (simple linear regression).ipynb
Performs simple linear regression analyses used in the manuscript, including annual female age composition, long-term age-group trends, annual water temperature trends,
female CPUE versus water temperature, regression diagnostics and residual analysis.

**Input:** `data.xlsx`

**Outputs:** publication figures, regression statistics, `female_age_summary.xlsx`
### Constructing_Leslie_Dominant_eigenvalue_SAD.ipynb
Constructs and analyzes the Leslie matrix used in the manuscript. The notebook estimates age-specific fecundities and survival probabilities, computes the dominant eigenvalue and stable age distribution, and compares the predicted stable age distribution with the observed female age composition.

**Input:** `data.xlsx`

**Output:** Leslie matrix, dominant eigenvalue, stable age distribution, and summary statistics.
### Sensitivity and Elasticity matrices.ipynb
Computes the sensitivity and elasticity matrices for the Leslie matrix. The notebook evaluates the influence of age-specific survival and fecundity parameters on the dominant population growth rate and produces annotated heatmaps together with the corresponding numerical matrices.

**Input:** Leslie matrix.

**Output:** Sensitivity and elasticity matrices (numerical values and annotated heatmaps).
### Impact_of_parameters_Scenarios.ipynb
Evaluates the effects of changes in age-specific fecundity and survival on the Leslie matrix model. Individual parameters are perturbed under predefined scenarios, and the resulting stable age distributions and population growth rates are compared with the baseline model.

**Input:** Leslie matrix.

**Output:** Scenario comparison figures and summary statistics.
